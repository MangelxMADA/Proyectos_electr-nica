# Portafolio de Electrónica ESP32 - Física 3

Bienvenido al repositorio del **Portafolio de Proyectos de Electrónica**, desarrollado en el marco de la asignatura de Física 3 (Oscilaciones, Ondas y Termodinámica) de la Universidad Tecnológica de Pereira. 

Este proyecto es una aplicación web interactiva (Single Page Application) construida con HTML, CSS y JavaScript que documenta de manera integral una serie de montajes prácticos realizados con el microcontrolador **ESP32**.

## 🎯 Objetivo del Proyecto

Más allá de los conceptos teóricos de la física, este portafolio se enfoca en el aprendizaje práctico de cómo diseñar, ensamblar y programar proyectos de electrónica. Abarca desde el control básico de componentes hasta la implementación de interfaces gráficas, comunicación inalámbrica (Bluetooth/WiFi) y telemetría IoT.

## 🛠️ Tecnologías y Herramientas

- **Frontend:** HTML5, CSS3 (Glassmorphism/Neumorphism), JavaScript Vanilla.
- **Simulaciones Físicas y Circuitos:** HTML5 Canvas API.
- **Microcontrolador:** ESP32 Dev Module.
- **Lenguaje embebido:** C/C++ (Arduino IDE).
- **Fórmulas y Ecuaciones:** KaTeX.
- **Asistente Virtual:** Integración de un Chatbot interactivo (ESP32Bot) especializado en hardware y depuración.

## 🚀 Proyectos Documentados

El portafolio incluye la documentación completa, esquemas, código fuente, videos demostrativos y simulaciones interactivas de los siguientes proyectos:

1. **On/Off Repetitivo:** Control básico de salidas digitales (Blink).
2. **On/Off por Teclado:** Comunicación serial bidireccional para el control de actuadores.
3. **Control con Potenciómetro:** Uso de ADC de 12 bits y Modulación por Ancho de Pulso (PWM).
4. **Potenciómetro con Bluetooth:** Transmisión de datos inalámbrica mediante Bluetooth Classic (SPP).
5. **Potenciómetro con WiFi:** Creación de un servidor web embebido y control IoT en red local.
6. **Potenciómetro con ThingSpeak:** Telemetría en la nube e Internet de las Cosas.
7. **Temperatura sin Termómetro (DHT22):** Lectura de sensores digitales para variables físicas.
8. **Temperatura con Termómetro (OLED):** Integración de pantallas gráficas y comunicación I2C.
9. **Movimiento Armónico Simple:** Aplicación de conceptos físicos con electrónica y visualización en tiempo real.
10. **Sensor de Distancia con Pantalla:** Uso de sensor ultrasónico HC-SR04 y pantalla OLED para instrumentación técnica.

## 📂 Estructura del Repositorio

- `index.html`: Estructura principal de la aplicación web (SPA) y contenido detallado de los proyectos.
- `styles.css`: Hojas de estilo que implementan un diseño moderno con efectos de glassmorphism, modo oscuro técnico y diseño responsivo.
- `app.js` / `script.js`: Lógica del frontend, navegación por secciones, visualización de imágenes interactivas y renderizado de simulaciones.
- `chatbot.js` y `chatbot.css`: Lógica y diseño del asistente virtual "ESP32Bot" integrado en el portafolio.
- `Arduino/`: Códigos fuente (`.ino`) para cada uno de los montajes documentados.
- `Esquemas/`: Diagramas de conexión y esquemáticos visuales de los circuitos.
- `Imagenes/` y `Videos/`: Recursos multimedia, fotografías del proceso de ensamble y demostraciones de funcionamiento.
- `Teoria proyectos/`: Informes técnicos en formato académico desarrollados con LaTeX.

## 💡 Interfaz y Diseño

El sitio está diseñado para proporcionar una experiencia de usuario envolvente con un **tema oscuro técnico**, priorizando la legibilidad de códigos y esquemas. Incluye un sistema de navegación lateral dinámico (*scroll-spy*), tablas de contenido interactivas, animaciones integradas y renderizado de ecuaciones matemáticas a nivel profesional.
